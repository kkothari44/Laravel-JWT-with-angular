For Backend
1.) Unzip the sample folder
2.) Opne sample folder after unzip
3.) Now install composer on your machine and Run "composer update"
4.) Create a database on mysql
5.) Open .env file and update your database credencial with db name
6.) Once updated run on root directory "php artisan migrate"
7.) After that run "php artisan serve"
8.) Application will start serving on port 8000
----------- Your backend servr is up------------

For Frontend

1.) Unzip the sampleFrontend folder
2.) Install Node js 
3.) Install Angular cli
4.) Run "nmp install"
5.) Once done installed run "nmp start"
6.) Application will start serving on port 8080
----------- Your fronted servr is up------------
